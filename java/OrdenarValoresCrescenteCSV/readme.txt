Para execução deste código é necessário que seja feito o download do arquivo que está nesta pasta chamado CVS.txt , uma vez que para realização do teste é necessário a leitura de um arquivo CSV com  os valores que serão ordenados em ordem crescente. É necessário também o mapeamento do arquivo dentro do código do main na linha 8 

String path = "caminho do arquivo ";

Exemplo : String path = "c:\\csv\\CVS.txt";

Após feito download e arquivo mapeado o código poderá rodar sem erros.
